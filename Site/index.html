Binary Calculator Done

The BinaryCalculator class is a utility for performing basic arithmetic operations—addition, subtraction, multiplication, and division—on binary numbers represented by the BitField class. The class uses bitwise operations to carry out these calculations, which is efficient for binary arithmetic. It uses only binary logic to complete these equations, as an excersize in understanding how binary numbers function. Here is a GitHub page for the code.

BeReal Project Done

I was taking a course on Human Computer Interaction, and the final assignment was to find UX design flaws in a popular website or app, and otherwise figure out how to modify a popular website or app in ways that would improve it for users, and possibly make it more popular. I chose BeReal because I felt like it was an app that was extremely popular at one point, but had been expiriencing a slow decline, so it was a fun challenge to pick BeReal and try to figure out what changes could be made to increase consumers regular use of the app. This project involved interviewing, creating flow models, WAAD, prototyping, and creating usage scenarios, among other things. You can take a look for youself here.
